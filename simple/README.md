# 📚 图书馆管理系统

一个基于 Flask 的图书馆管理系统，支持图书借阅、读者管理和管理员操作等功能。

## ✨ 功能特性

### 读者功能
- 📝 用户注册与登录
- 📖 浏览可借阅的图书和期刊
- 📚 借阅图书（最多3本）
- 🔄 归还图书
- 📊 查看个人借阅记录
- ⏰ 查看借阅到期时间

### 管理员功能
- 🔐 管理员登录（默认账号：admin / admin123）
- ➕ 添加图书和期刊
- 📋 查看所有出版物列表
- 👥 查看所有读者信息
- 📊 管理借阅状态

### 系统特性
- 📖 支持图书和期刊两种出版物类型
- ⏱️ 图书借阅期限：14天
- 📰 期刊借阅期限：最新期刊7天，过刊14天
- 💾 数据持久化（JSON文件存储）
- 🎨 现代化响应式界面设计

## 🚀 快速开始

### 环境要求
- Python 3.7+
- Flask 3.0.0

### 安装步骤

1. 克隆或下载项目到本地

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行应用
```bash
python app.py
```
或在 Windows 上双击运行：
```bash
run.bat
```

4. 访问应用
打开浏览器访问：`http://127.0.0.1:5000`

## 📖 使用说明

### 管理员登录
- 用户ID：`admin`
- 密码：`admin123`

### 读者注册
1. 点击"登录系统"
2. 点击"立即注册"
3. 填写姓名、读者ID和密码
4. 注册成功后使用读者ID和密码登录

### 借阅图书
1. 以读者身份登录
2. 在"可借阅图书"列表中选择图书
3. 点击"借阅"按钮
4. 系统会显示归还日期

### 归还图书
1. 在"我的借阅"列表中找到要归还的图书
2. 点击"归还"按钮

### 添加图书（管理员）
1. 以管理员身份登录
2. 在"添加图书"表单中填写信息
3. 点击"添加图书"按钮

## 🏗️ 核心类设计

### Publication（出版物基类）
- 管理借阅状态
- 处理借阅和归还消息
- 定义最大借阅天数

### Book（图书类）
- 继承自 Publication
- 包含作者、ISBN、分类信息
- 借阅期限：14天

### Magazine（期刊类）
- 继承自 Publication
- 包含期号、出版商信息
- 支持最新期刊和过刊标记
- 借阅期限：最新期刊7天，过刊14天

### Library（图书馆类）
- 管理所有出版物和读者
- 提供权限控制
- 处理借阅业务逻辑

### Reader（读者类）
- 管理个人借阅记录
- 借阅限额：3本
- 发送借阅和归还消息

### Admin（管理员类）
- 添加/删除出版物
- 注册读者
- 管理图书馆资源

## 💾 数据存储

系统使用 JSON 文件（`library_data.json`）存储数据，包括：
- 读者信息（姓名、ID、密码、借阅限额）
- 出版物信息（图书和期刊的详细信息）

数据在以下操作后自动保存：
- 读者注册
- 添加图书
- 借阅/归还操作
